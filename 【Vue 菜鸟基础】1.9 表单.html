1.9 表单

  • 输入框
  • 单选
  • 复选
  • 选择列表
  • 修饰符
  • v-model 实现数据双向绑定；

输入框

<div id="app">
    输入框<input v-model="name">
    <p>{{name}}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            name: ''
        }
    });
</script>

单选

    男<input type="radio" value="男" v-model="sex">
    女<input type="radio" value="女" v-model="sex">
    <p>{{sex}}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            sex: ''
        }
    });
</script>

复选

  • 单个复选 true or false；
  • 多个复选 数组；
<div id="app">
    我已阅读<input type="checkbox" value="read" v-model="read">
    <p>{{read}}</p>


    语文<input type="checkbox" value="语文" v-model="class">
    数学<input type="checkbox" value="数学" v-model="class">
    <p>{{class}}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            read:false,
            class: ['数学']
        }
    });
</script>

选择列表

<div id="app">
    <select v-model="lesson" name="A123">
        <option value="1" selected>语文</option>
        <option value="2">数学</option>
    </select>

    <p id="output">{{lesson}}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            lesson: '1'
        }
    });
</script>

修饰符

.lazy   change 事件，而不是输入变化
.number 转换为 Number
.trim   去除前后空格
