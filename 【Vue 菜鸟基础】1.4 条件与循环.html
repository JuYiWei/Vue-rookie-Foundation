1.4 条件与循环

  • 条件
  • 显示与隐藏
  • 循环

条件

  • v-if、v-else-if、v-else；
<div id="app" @click="randomAgain">
    <div v-if="r < 60">不及格</div>
    <div v-else-if="(r >= 60) && (r < 80)">良好</div>
    <div v-else="r >= 80">优秀</div>
</div>


<script>
    var app = new Vue({
        el: '#app',
        data: {
            r: 80
        },
        methods: {
            randomAgain: function () {
                this.r = Math.random()*100;
            }
        }
    })
</script>

显示与隐藏

  • v-show；
<div id="app" @click="hide">
    <div v-show="look">点击隐藏</div>
</div>


<script>
    var app = new Vue({
        el: '#app',
        data: {
            look: true
        },
        methods: {
            hide: function () {
                this.look = false;
            }
        }
    })
</script>

循环

  • 可以遍历数组（特殊字典）、字典、数值；
  • 可以多参数；
<div id="app">
    <div v-for="obj in books">
        {{obj}}
    </div>

    <div v-for="obj in object">
        {{obj}}
    </div>

    <div v-for="(obj, key, idx) in object">
        {{obj}} - {{key}} - {{idx}}
    </div>
    
    <div v-for="n in 10">
        {{n}}
    </div>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            books: [
                '语文', '数学'
            ],
            object: {
                a: 'aaa',
                b: 'bbb'
            }
        }
    })
</script>
