1.3 模板语法

  • 插值
    • 文本 {{}}
    • html v-html
    • 属性 v-bind
    • 表达式
  • 指令
    • 一般指令
    • 参数
    • 修饰符
  • 数据双向绑定
  • 过滤器
  • 缩写

插值

文本 {{}}

{{ ... }} 双大括号绑定数据

<div id="app">
  <p>{{ message }}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue!',
        }
    })
</script>

html v-html

html 中 {{ ... }} 不能用，但可用 " ... " 或 ' ... ' 替代；
"' xxx '" 或者 '" xxx "' 实现非绑定数据；

<div id="app">
    <div v-html="message"></div>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            message: '<p style="color: red">Hello Vue!</p>'
        }
    })
</script>

属性 v-bind

    <style>
        .testClass {
            background: red;
        }
    </style>

<div id="app">
    <div v-bind:id="messageId" v-bind:class="'testClass'">message</div>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            messageId: 'test-id'
        }
    })
</script>

表达式

  • 在 数据绑定形式中，可使用任意 JavaScript 表达式；
<div id="app">
    <div v-bind:id="'id-'+(1+4)">{{ 1 + 3 }}</div>
</div>

指令

v- 前缀的特殊属性；

一般指令

<div id="app">
    <div v-if="index > 3">目标 大于 3</div>
    <div v-if="index > 10">目标 大于 10</div>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            index: 4
        }
    })
</script>

v-if 就是指令；

参数

<div id="app">
    <div v-bind:id="'messageId'">message</div>
</div>

:id 表示绑定 id 属性；

修饰符

<form v-on:submit.prevent="onSubmit"></form>

.prevent 修饰符，表示 v-on 触发的事件调用 event.preventDefault()；

数据双向绑定

<div id="app">
    <input v-model="message">
    <p>{{ message }}</p>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            message: 4
        }
    })
</script>

过滤器

  • {{ param | function }} 的形式，添加过滤器；
  • {{ param | filterA | filterB }} 串联；
  • {{ param | filterA('param2', param3) }} 添加多参数；
<div id="app">
    {{ name | nameHandle }}

</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            name: 'li lei'
        },
        filters: {
            nameHandle: function (value) {
                if (!value) return ''
                value = value.toString()
                return value.charAt(0).toUpperCase() + value.slice(1)
            }
        }
    })
</script>

缩写

  • v-bind: 缩写为 :；
  • v-on: 缩写为 @
<div id="app">
    <div :id="'test-id'" @click="testClick">缩写</div>
</div>

<script>
    var app = new Vue({
        el: '#app',
        data: {
            name: 'li lei'
        },
        methods: {
            testClick: function () {
                alert('123');
            }
        }
    })
</script>
